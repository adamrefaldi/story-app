# Informasi Proyek Dicoding Story Platform

## Keterangan
- Proyek ini menggunakan OpenStreetMap yang tidak memerlukan API key
- URL API yang digunakan: https://story-api.dicoding.dev/v1

## Fitur
- Login/Register
- Menampilkan daftar cerita
- Menambahkan cerita baru dengan foto dan lokasi
- Melihat lokasi cerita pada peta
- Pemilihan lokasi pada peta
- Pengambilan foto menggunakan kamera
- Aksesibilitas dengan skip to content, label, dan semantic elements
- Responsive design
- Transisi halaman yang halus dengan View Transition API
- Multiple layer map control

## Sumber
Proyek ini dibuat untuk memenuhi kriteria submission Dicoding.
